Zadanie 8. (0–1 pkt)

Dokończ zdanie. Wybierz właściwą odpowiedź spośród podanych.

\[ \frac{(x+1)(x-1)^2}{(x-1)(x+1)^2}=0 \]

Równanie w zbiorze liczb rzeczywistych:

  • nie ma rozwiązania,
  • ma dokładnie jedno rozwiązanie: \( -1 \),
  • ma dokładnie jedno rozwiązanie: \( 1 \),
  • ma dokładnie dwa rozwiązania: \( -1 \) oraz \( 1 \).
Pokaż rozwiązanie

Rozwiązanie (zgodne z kryteriami oceniania CKE)

Krok 1. Dziedzina

Mianownik nie może być równy zero:

\[ (x-1)(x+1)^2 \neq 0 \]

Stąd: \[ x \neq 1 \quad \text{oraz} \quad x \neq -1 \]

Krok 2. Warunek zerowania się ułamka

Ułamek jest równy zero wtedy i tylko wtedy, gdy licznik jest równy zero oraz mianownik jest różny od zera.

\[ (x+1)(x-1)^2 = 0 \]

Potencjalne rozwiązania:

  • \(x=-1\)
  • \(x=1\)

Krok 3. Weryfikacja z dziedziną

  • \(x=-1\) – nie należy do dziedziny
  • \(x=1\) – nie należy do dziedziny

Wniosek

Równanie nie ma rozwiązań w zbiorze liczb rzeczywistych.

Odpowiedź poprawna: A

Zadanie 10. (0-1 pkt)

Treść zadania

Dokończ zdanie. Wybierz właściwą odpowiedź spośród podanych. Równanie \[ \frac{(x^2 - 3x)(x+2)}{x^2 - 4} = 0 \] w zbiorze liczb rzeczywistych ma dokładnie:

  • jedno rozwiązanie
  • dwa rozwiązania
  • trzy rozwiązania
  • cztery rozwiązania
Pokaż rozwiązanie

Rozwiązanie (zgodne z kryteriami oceniania CKE)

Krok 1. Dziedzina

Mamy równanie wymierne, więc musimy wykluczyć wartości, dla których mianownik jest równy zero:

\[ x^2 - 4 = 0 \implies x = 2 \text{ lub } x = -2 \]

Zatem dziedziną równania jest:

\[ D = \mathbb{R} \setminus \{-2, 2\} \]

Krok 2. Rozwiązanie równania

Równanie wymierne jest równe zero wtedy i tylko wtedy, gdy licznik jest równy zero (i liczba należy do dziedziny):

\[ (x^2 - 3x)(x+2) = 0 \]

Rozwiązujemy równanie iloczynowe:

\[ x^2 - 3x = 0 \quad \text{lub} \quad x + 2 = 0 \] \[ x(x - 3) = 0 \implies x = 0 \text{ lub } x = 3 \] \[ x + 2 = 0 \implies x = -2 \]

Krok 3. Weryfikacja

Sprawdzamy, które z tych rozwiązań należą do dziedziny \(D\): \[ x = 0 \in D, \quad x = 3 \in D, \quad x = -2 \notin D \]

Wniosek

Rozwiązania równania w zbiorze liczb rzeczywistych to \(x = 0\) oraz \(x = 3\). Równanie ma dokładnie dwa rozwiązania.

Odpowiedź poprawna: B

Zadanie 8 (0–1)

Treść zadania

Dokończ zdanie. Wybierz właściwą odpowiedź spośród podanych. Równanie \[ \frac{(x^2 - 3x)(x^2 + 1)}{x^2 - 25} = 0 \] w zbiorze liczb rzeczywistych ma dokładnie:

  • jedno rozwiązanie
  • dwa rozwiązania
  • trzy rozwiązania
  • cztery rozwiązania
Pokaż rozwiązanie

Rozwiązanie

Krok 1. Dziedzina

Wykluczamy wartości, dla których mianownik jest równy zero:

\[ x^2 - 25 = 0 \implies x = 5 \text{ lub } x = -5 \] \[ D = \mathbb{R} \setminus \{-5, 5\} \]

Krok 2. Rozwiązanie równania

Równanie jest równe zero, gdy licznik jest równy zero:

\[ (x^2 - 3x)(x^2 + 1) = 0 \] \[ x^2 - 3x = 0 \implies x(x - 3) = 0 \implies x = 0 \text{ lub } x = 3 \] \[ x^2 + 1 = 0 \implies x^2 = -1 \text{ – brak rozwiązań rzeczywistych} \]

Wniosek

Rozwiązania równania w zbiorze liczb rzeczywistych to \(x = 0\) i \(x = 3\). Równanie ma dokładnie dwa rozwiązania.

Odpowiedź poprawna: B

Zadanie 7 (0–1)

Treść zadania

Dokończ zdanie. Wybierz właściwą odpowiedź spośród podanych. Równanie \[ \frac{x+1}{(x+2)(x-3)} = 0 \] w zbiorze liczb rzeczywistych:

  • nie ma rozwiązania
  • ma dokładnie jedno rozwiązanie: \(-1\)
  • ma dokładnie dwa rozwiązania: \(-2\) oraz 3
  • ma dokładnie trzy rozwiązania: \(-1\), \(-2\) oraz 3
Pokaż rozwiązanie

Rozwiązanie

Krok 1. Dziedzina

Wykluczamy wartości, dla których mianownik jest równy zero:

\[ (x+2)(x-3) = 0 \implies x = -2 \text{ lub } x = 3 \] \[ D = \mathbb{R} \setminus \{-2, 3\} \]

Krok 2. Rozwiązanie równania

Równanie wymierne jest równe zero, gdy licznik jest równy zero:

\[ x + 1 = 0 \implies x = -1 \]

Wniosek

Rozwiązaniem równania w zbiorze liczb rzeczywistych jest \(x = -1\). Równanie ma dokładnie jedno rozwiązanie.

Odpowiedź poprawna: B

Matura CKE – 2024, sierpień, poziom podstawowy
Zadanie 6 (0–1)

Treść zadania

Dokończ zdanie. Wybierz właściwą odpowiedź spośród podanych. Równanie \[ \frac{x(x+5)(2-x)}{2x+4} = 0 \] w zbiorze liczb rzeczywistych ma dokładnie:

  • dwa rozwiązania: \(-5\) oraz 2
  • dwa rozwiązania: \(-5\) oraz 0
  • trzy rozwiązania: \(-5\), 0 oraz 2
  • cztery rozwiązania: \(-5\), \(-2\), 0 oraz 2
Pokaż rozwiązanie

Rozwiązanie

Krok 1. Dziedzina

Wykluczamy wartości, dla których mianownik jest równy zero:

\[ 2x + 4 = 0 \implies x = -2 \] \[ D = \mathbb{R} \setminus \{-2\} \]

Krok 2. Rozwiązanie równania

Równanie wymierne jest równe zero, gdy licznik jest równy zero:

\[ x(x+5)(2-x) = 0 \] \[ x = 0, \quad x + 5 = 0 \implies x = -5, \quad 2 - x = 0 \implies x = 2 \]

Wniosek

Rozwiązania w dziedzinie to \(x = -5\), \(x = 0\) oraz \(x = 2\). Równanie ma dokładnie trzy rozwiązania.

Odpowiedź poprawna: C

Zadanie 8 (0–3)

Treść zadania

Dane jest równanie \[ \frac{3}{3x - 7} = \frac{5x}{x - 8}, \quad x \neq \frac{7}{3}, \; x \neq 8 \] Wyznacz wszystkie rozwiązania tego równania należące do przedziału \((\frac{5}{4}, +\infty)\). Zapisz obliczenia.

Pokaż rozwiązanie

Rozwiązanie

Krok 1. Dziedzina

Podano ograniczenia: \(x \neq \frac{7}{3}\) oraz \(x \neq 8\). Zatem dziedziną równania jest:

\[ D = \mathbb{R} \setminus \left\{\frac{7}{3}, 8\right\} \]

Krok 2. Przekształcenie równania

Mnożymy obustronnie przez \((3x-7)(x-8)\) (dopuszczalne dla \(x \in D\)):

\[ 3(x - 8) = 5x(3x - 7) \] \[ 3x - 24 = 15x^2 - 35x \] \[ 0 = 15x^2 - 35x - 3x + 24 \] \[ 15x^2 - 38x + 24 = 0 \]

Krok 3. Rozwiązanie równania kwadratowego

Delta:

\[ \Delta = (-38)^2 - 4 \cdot 15 \cdot 24 = 1444 - 1440 = 4 \] \[ x = \frac{38 \pm \sqrt{4}}{2 \cdot 15} = \frac{38 \pm 2}{30} \] \[ x_1 = \frac{40}{30} = \frac{4}{3}, \quad x_2 = \frac{36}{30} = \frac{6}{5} \]

Krok 4. Sprawdzenie rozwiązań z dziedziną

Oba rozwiązania należą do dziedziny, żadne nie jest równe \(\frac{7}{3}\) ani 8. Sprawdzamy, które rozwiązania należą do przedziału \((\frac{5}{4}, +\infty)\): \(\frac{6}{5} = 1.2 < 1.25 \notin (\frac{5}{4}, +\infty)\), \(\frac{4}{3} \approx 1.333 > 1.25 \in (\frac{5}{4}, +\infty)\)

Wniosek

Rozwiązaniem równania w przedziale \((\frac{5}{4}, +\infty)\) jest:

x = \(\frac{4}{3}\)
